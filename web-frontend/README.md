# Twitter Alternative - Web Frontend

## Overview
Simple web UI for the Twitter Alternative social media platform built with vanilla HTML/CSS/JavaScript.

## Setup & Running

### Prerequisites
- Backend server running on `http://localhost:8080`
- Any modern web browser
- Python (for simple HTTP server - optional)

### Method 1: Python HTTP Server
```bash
cd web-frontend
python -m http.server 3000
# Visit http://localhost:3000
